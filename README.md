# README - Web Scraping em Python 

## Introdução 
Este projeto é uma implementação de web scraping em Python. Utiliza as bibliotecas requests para realizar requisições HTTP e lxml para analisar o conteúdo HTML da página da web. O objetivo é extrair informações específicas de um site e salvá-las em um formato estruturado, como JSON.

## Requisitos

- Python 3.x
- Bibliotecas: requests, lxml

## Instalação
 ```python
pip install requests lxml
```
